# OSARK Server

## Description

OSARK Server is a server that captures events from the OSARK client and stores them in a database.

## Installation

```bash
go mod tidy
```

## Usage

```bash
docker compose up --build -d
```

To stop the server, run:

```bash
docker compose down
```
